# MorphPadala

MorphPadala is a cross-border remittance prototype built for the Morph Build In! Payments Hackathon.

## Features
- Cross-border remittance simulation
- Real-time FX conversion
- Receiver wallet visualization
- Instant settlement flow
- OFW-focused remittance experience

## Tech Stack
- React
- Vite
- TailwindCSS

## Use Case
MorphPadala helps Overseas Filipino Workers send money home faster and cheaper using Morph-powered payment infrastructure.
